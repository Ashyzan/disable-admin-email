Turns off the notification sent to the admin email when a new user account is registered.

== Description ==

A simple plugin that disables the notification email sent to the admin, only when a new user account is registered. Are you tired of receiving new emails for every user who registers on your site? Yes, it's boring, especially if you are the administrator of a site that allows user registration and you receive many emails a day. Sending a new user registration email is a basic WordPress feature, which aims to ensure security, however you would like to be able to disable these notifications but you don't know how. Here's a quick and easy solution!
For use it, simple activate it! You will no longer receive registration emails from new users, while they (users) will regularly receive their site registration email.

If you like this plugin, please consider leaving a comment or review. 


== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
2. Search for "Disable Admin Email".
3. Click "Install Now".
4. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
